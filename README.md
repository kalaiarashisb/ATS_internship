# Titanic Survival Prediction using Decision Tree

## Project Overview
This project predicts passenger survival on the Titanic dataset using a Decision Tree Classifier.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

## Features Used
- Sex(male,female)
- Age
- Passenger Class

## Machine Learning Model
- Decision Tree Classifier

## Steps Performed
1. Data preprocessing
2. Handling missing values
3. Label encoding
4. Train-test split
5. Model training
6. Prediction and evaluation
7. Decision tree visualization

## Accuracy
Achieved approximately 81% accuracy.

## Key Insights
- Female passengers had higher survival rates.
- First and second class passengers survived more.
- Younger passengers had better survival chances.
